## 주요 문제점 3가지

### 1. 가이드 템플릿 중복 (가장 큰 위험)
- 80개 가이드 중 JSON 데이터 없는 페이지는 **동일한 폴백 템플릿** 사용
- 구글이 "중복된 낮은 품질 콘텐츠"로 판단 가능

### 2. Canonical URL 누락
- 모든 페이지에 canonical 태그 없음
- Vercel 서브도메인(phiennoi-platform.vercel.app)과 커스텀 도메인(vn.epicstage.co.kr) 중복 색인 위험

### 3. 콘텐츠 깊이 편차
- 용어 meaningKo 목표 150자이나 실제 60~80자 수준인 용어 약 20%
- pronunciation, commonMistakes, culturalNote 필드 대부분 미활용

## 우선순위별 개선안

### P0 (즉시)
1. Canonical URL 추가
2. JSON 없는 가이드 noindex 처리
3. OG 이미지 + Twitter Card 메타태그

### P1 (1주)
4. 용어 meaningKo 150자 미만 → 보강

7. 핵심 가이드 20개 고유 콘텐츠 작성
8. 한자 분해 SVG 시각화
9. 사용자 피드백 시스템
